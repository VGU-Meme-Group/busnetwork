# How to run the Bus Application

We use 2 communication APIS in this project :
- REST Based Communication APIs
- Web Socket Based Communication APIs

## Setup the Server

- Install the environment
```bash
npm install
```
- Run the REST Server
```bash
npm start
```
- Run the Web Socket Server
```bash
npm run socket
```

## Setup the Frontend 
- Run the Frontend 
```bash
npm run dev
```
